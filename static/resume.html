03/2017-07/2018
 
Automatic Return System for Library Books Based on RFID and AGV Technologies
 
Research Project in High School
 
This project aims to improve efficiency by saving time and power through the introduction of a library book return system that combines RFID and AGV technology. This system can acquire book location information and transport books to the corresponding bookshelf. The location information of books is stored in RFID electronic tags, which are attached to the inside of books. To transport the book to the corresponding bookshelf, the book's location information is transferred to the vehicle (lower unit) using upper unit BLE technology. The vehicle uses Hall sensors to transport the books from the end of the conveyor to the corresponding stacks using magnetic navigator (AGV) technology. To visualize the operation status, a screen was created using VB, utilizing an Access database.

Yi DianDian (One ponit Billion points)
 
Innovation& Entrepreneurship Training Program for College Students-Shaanxi Provincial Project
 
Research has shown that learning before bedtime significantly improves memory effectiveness; information heard before bedtime remains in the brain unconsciously and is reproduced repeatedly. This project will utilize PPG sensors and inertial sensors, which are widely used in smartwatches and wearable devices, to acquire real-time sleep information from the user and realize multimedia playback and pause. As the user prepares for sleep, it can play text that circulates the desired memory material. This can assist the user in remembering necessary information. When the smart device detects that the user has entered sleep, it will automatically pause and record the pause point, allowing playback to continue from that point the next time it is used.
 
10/2020-12/2020
 
AI Medical CAM
 
Internet+ Competition
 
The project aims to provide rural residents in China with advice and solutions for skin surface conditions that arise in the course of farming and daily life. Users simply photograph symptomatic skin areas with their smartphones, select the skin areas photographed in the mini-program, and upload these data to a server. Cloud-based multimodal deep learning calculations provide the probability of the skin disease the user is suffering from and corresponding advice, which is eventually back to the user via the mini-program to facilitate easy processing and early medical attention for the user. Skin disease identification is divided into lesion segmentation and classification. Segmentation is done using Mask-RCNN and lesion classification using the Inception-V4 network.
